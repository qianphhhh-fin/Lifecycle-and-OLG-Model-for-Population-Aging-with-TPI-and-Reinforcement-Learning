本仓库包含了清华大学经济学博士学位论文《人口老龄化背景下我国多层次养老保障体系与居民养老问题研究》的实证代码与模型实现。

本研究构建了一个从微观家庭决策到宏观一般均衡的完整量化分析框架，旨在系统评估人口老龄化背景下，中国养老金体系改革（如个人养老金制度、延迟退休）的宏观经济效应与微观福利影响。

## 📂 代码结构 (Code Structure)

代码仓库主要依据论文的章节逻辑进行组织，涵盖了从微观生命周期模型到宏观 OLG 模型的实现。

> **注意**：具体文件名可能随版本更新略有调整，请以仓库实际文件为准。

### 1. 微观基础：个人养老金决策 (对应论文第 4 章)
*   **路径**: `/1_养老金基准`
*   **核心内容**: 包含内生性个人养老金（Private Pension System, PPS）购买决策的基准生命周期模型。
*   **算法**: 逆向归纳法 (Backward Induction) + 网格搜索 (Grid Search)。
*   **功能**: 求解在税收激励、流动性约束和预防性储蓄动机下的最优消费、投资与养老金缴费策略。

### 2. 结构异质性：城乡二元结构下的资产配置 (对应论文第 5 章)
*   **路径**: `/2_养老金税收拓展`
*   **核心内容**: 嵌入城乡二元结构（城镇职工 vs. 城乡居民）的扩展生命周期模型。
*   **算法**: **深度强化学习 (Deep Reinforcement Learning)** - Soft Actor-Critic (SAC)。
*   **亮点**: 解决了包含多维连续状态变量（财富、收入冲击、养老金余额、缴费年限）和连续决策变量的高维动态优化问题，克服了传统动态规划的“维度诅咒”。
*   **依赖**: `PyTorch`, `NumPy`.

### 3. 宏观均衡：动态世代交叠模型 (对应论文第 6 章)
*   **路径**: `/3_OLG`
*   **核心内容**: 包含四类异质性代理人（高/低收入 × 城镇/农村）的大规模世代交叠（OLG）一般均衡模型。
*   **算法**: 时序路径迭代法 (Time Path Iteration, TPI)。
*   **功能**:
    *   求解长期平衡增长路径 (BGP) 稳态。
    *   模拟从当前状态向未来稳态过渡的完整动态路径 (Transition Dynamics)。
    *   评估延迟退休、个人养老金推广等政策的代际福利影响。
*   **依赖**: `MATLAB` (主要涉及非线性方程组求解 `lsqnonlin`).

## 📖 论文介绍 (Dissertation Overview)

### 摘要 (Abstract)
在人口结构快速老龄化的宏观背景下，我国养老金体系面临“第一支柱独大、第三支柱滞后”的结构性失衡。本研究构建了一个将微观家庭异质性行为与宏观经济动态有机整合的量化分析框架。

研究首先构建微观生命周期模型，揭示了个人养老金缴费的“驼峰形态”及流动性约束的扭曲效应；进而引入城乡二元结构，利用强化学习算法求解发现现行税收优惠政策存在显著的“累退性”，对农村居民激励不足；最后，通过构建包含四类异质性家庭的动态 OLG 模型，量化了延迟退休改革的代际成本，并指出单纯依赖个人养老金不足以弥补现收现付制（PAYG）的缺口。

### 核心模型与方法 (Methodology)

1.  **微观决策模型 (Micro-foundation)**:
    *   采用 Epstein-Zin 递归效用函数。
    *   内生化个人养老金参与、风险资产配置与消费决策。
    *   精细刻画中国特有的“EET”税收递延机制与账户锁定约束。

2.  **强化学习求解器 (DRL Solver)**:
    *   针对第 5 章的高维状态空间（财富、养老金账户、累积缴费年限、收入冲击等），创新性引入 **Soft Actor-Critic (SAC)** 算法。
    *   利用深度神经网络逼近策略函数和价值函数，实现了对复杂约束下家庭最优决策的高效求解。

3.  **宏观 OLG 与 TPI 算法**:
    *   构建 80 期生存周期的 OLG 模型，内嵌 PAYG 和 PPS 双支柱体系。
    *   使用 **Time Path Iteration (TPI)** 算法求解经济体在人口冲击下的非稳态过渡路径，捕捉“过渡代际”的福利损失。

### 主要结论 (Key Findings)

1.  **微观激励与流动性约束**:
    *   个人养老金缴费呈现显著的生命周期“驼峰”特征。
    *   现行缴费上限对职业中期群体造成了显著的福利扭曲。

2.  **城乡不平等与政策累退性**:
    *   城镇居民能有效利用税收优惠，而农村居民因收入波动大、纳税基数低，对税收激励不敏感。
    *   现行个人养老金政策在城乡间表现出显著的累退性（Regressivity）。

3.  **宏观改革的代际权衡**:
    *   延迟退休能有效缓解财政压力，但成本主要由临近退休、缺乏调整时间的“过渡代际”承担。
    *   个人养老金与预防性储蓄叠加，可能引致宏观层面的资本过度积累（动态无效率）。
